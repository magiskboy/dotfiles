### Pre-requirement

In Ubuntu

```bash
$ sudo apt install git
```

In MacOS
```bash
$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
$ brew install git
```

### How to install

```bash
$ git clone https://github.com/magiskboy/dotfiles $HOME/dotfiles
$ cd $HOME/dotfiles
$ ./install.sh base
```
